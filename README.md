## 社团信息

- 社团名称：浙江大学国际校区学生书香阅读社
- 社团英文名：iZJU Reading Club
- 社团活动室：惟学106阅览室（每学年与惟学书院方面确认。当前联系人：刘娜）
- 社团定位：提供阅读爱好者交流、活动的平台，并鼓励校区阅读文化的成长
- 社团品牌活动：回归读书会，周六共读时间，月度读书会

### 社团活动介绍

读书会0

- 时间：寒暑假结束后20天内，具体时间视具体情况而定

- 地点：图书馆多功能厅

- 书目：寒/暑假共读书目

  大型读书会。期待十数人的参与。一般而言，九月的暑假回归读书会将会期待10-20人的实际参与，寒假回归读书会将会期待4-10人的实际参与。活动名称采用“书名+读书会”格式（i.e.“读书会0”为社团内部记录名称）

月度读书会

- 时间：每月月底，如有时间冲突顺延。保持~1月的活动间隔。

- 地点：惟学106阅览室

- 书目：主讲人决定，不违反核心价值观即可

  小型读书会。

纪念读书会

- 时间：纪念日/节日（例：中国记者节，女生节，etc.）
- 地点：惟学106阅览室/任意合适位置。关注申请流程以及合规性。
